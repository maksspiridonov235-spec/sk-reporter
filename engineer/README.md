# Инженер ФИО — данные в репозитории

Раздел **Инженер ФИО** в UI: `/engineer-hub` → `/engineer/{profile_id}`.

| Папка | Назначение |
|-------|------------|
| `profiles/` | yaml-профили (person_id, проекты, шаблон docx) |

## Как появляется профиль

1. В **Планирование → Проекты** назначают инженера на объект.
2. Создаётся `profiles/{id}.yaml` (если ещё нет).
3. Инженер открывает карточку на https://sk-reporter.relaxdev.ru/engineer/{id} → заполняет ВОР → скачивает docx.

Шаблон отчёта: `data/engineer/report_template.docx` (положить вручную).
