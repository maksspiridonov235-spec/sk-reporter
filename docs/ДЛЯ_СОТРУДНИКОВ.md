# SK-Reporter — для сотрудников

Ничего устанавливать не нужно. Работа только в браузере.

## Как открыть

1. Ответственный дважды нажимает ярлык **SK-Reporter** на рабочем столе (или запускает сервер).
2. Открывается браузер: http://localhost:8000
3. Чёрное окно консоли **не закрывать** — там работает сервер.

## Что делать на странице

1. Загрузить отчёты (.docx) — перетащить или нажать на зону загрузки.
2. Выбрать **дату** в поле под списком файлов.
3. При необходимости: **Проверить и исправить**.
4. **Подготовить** → **Сформировать**.
5. Скачать готовые файлы в правой части экрана или **Скачать (ZIP)**.

## Если не открывается

- Спросить ответственного: запущен ли ярлык SK-Reporter.
- Ollama должна быть в трее (иконка у часов) — нужна для кнопок с AI.
