## ▶ СЕЙЧАС В РАБОТЕ (читать первым)

**Дата:** 2026-06-04  
**Фокус сессии:** **оба блока в сайдбаре UI — блок 1 и блок 2.** Пользователь явно сказал: работаем с **двумя** блоками (скрины в чате от 2026-06-04).

Новый чат — продолжение **без повторного объяснения**. Оба блока в scope, пока пользователь не сузит задачу.

### Блок 1 — «Проверить и исправить»

| UI | Код / API |
|----|-----------|
| **Проверить и исправить** | `#btnCheck` → `startCheck()` → `POST /check/descriptions/stream` |

Цепочка: `check_agent` (Ollama) → `inject_agent` → **запись в загрузку** (`UPLOAD_DIR`, то же имя файла). Скачать с именем `{имя}_исправлен.docx` — `/download/fixed/...` (читает тот же файл из загрузки).

| Слой | Файл |
|------|------|
| UI | `webapp/templates/index.html`, `webapp/static/app.js` (`startCheck`) |
| API | `webapp/main.py` — stream check, `/download/fixed/...` |
| AI + inject | `sk_reporter/agent/check_agent.py`, `sk_reporter/agent/inject_agent.py` |
| Загрузка (сырые + после проверки) | temp `sk_reports_work/uploads/` — см. `UPLOAD_DIR` в `webapp/main.py` |

Важно: inject пишет в ячейку-заголовок **«Описание действий»**; после проверки перезаливать не нужно. Отдельных папок `output/` в репо нет.

### Блок 2 — Аниськов / Манджиев (руководитель в шапке)

| Кнопка | Действие |
|--------|----------|
| **Аниськов В.И.** (синяя) | `switchLeader('aniskov')` |
| **Манджиев И.А.** (оранжевая) | `switchLeader('mandzhiev')` |

По клику: для **всех загруженных** `.docx` в `UPLOAD_DIR` — поиск и замена ФИО руководителя (Ollama).

| Слой | Файл |
|------|------|
| UI | `webapp/templates/index.html`, `webapp/static/app.js` → `switchLeader()` |
| API | `webapp/main.py` → `POST /switch-leader-ai/{leader}` |
| AI | `sk_reporter/agent/leader_ai_agent.py` |

---

### Как работать агенту в новом чате

1. Прочитать этот раздел целиком (оба блока).
2. Уточнить **конкретную задачу** по блоку 1, блоку 2 или обоим — если пользователь не написал в первом сообщении.
3. Не уводить в VPS / 20 bat / таблицу-форму без запроса (фон — `PROJECT_CONTEXT.md`).
4. Коммит/push — только по явной просьбе.
