```
загрузка → [дата] [руководитель] → Проверить и исправить
```

### Как работать агенту в новом чате

1. Прочитать **«СЕЙЧАС В РАБОТЕ»** целиком.
2. **Verify** — не делать и не восстанавливать из git без явного запроса владельца.
3. Блок 1 стабилен: `check_agent` → `inject_agent`; не ломать UI логов и не трогать `check_agent` без необходимости.
4. ZIP: **Скачать (ZIP)** — `отчеты/` = `RESULT_DIR` (сформированные); `исправленные/` = `UPLOAD_DIR` (`*_исправлен.docx`).
5. Не уводить в VPS / 20 bat / таблицу-форму без запроса (фон — `PROJECT_CONTEXT.md`).
6. Коммит/push — только по явной просьбе.
7. После `git pull` на офисном ПК — перезапустить bat.
