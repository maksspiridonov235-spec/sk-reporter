ЗАПУСК ПРИЛОЖЕНИЯ
=================

1. Установить зависимости (один раз):

   pip install -r requirements.txt

2. Запустить сервер:

   cd webapp
   python3 -m uvicorn main:app --reload --port 8000

   (если uvicorn не в PATH — всегда используй python3 -m uvicorn)

3. Открыть в браузере:

   http://localhost:8000


РАБОТА С ПРИЛОЖЕНИЕМ
====================

1. Загрузить отчёты от инженеров (левая панель, верхняя зона)
2. Загрузить шаблоны-болванки подрядчиков (левая панель, нижняя зона)
3. При необходимости применить макросы к шаблонам
4. Нажать кнопку компании для формирования одного отчёта
   ИЛИ "Сформировать все отчёты" для всех компаний сразу
5. Скачать готовые файлы из блока "Готовые файлы"


ДОБАВИТЬ НОВУЮ КОМПАНИЮ
=======================

Открыть webapp/docx_processing.py, найти список COMPANIES и добавить строку:

    ("НазваниеКомпании", ["ключевое слово1", "ключевое слово2"]),

Больше ничего менять не нужно — кнопка появится автоматически.


СТРУКТУРА ФАЙЛОВ
================

webapp/
  main.py              — FastAPI сервер
  docx_processing.py   — вся логика (макросы, merge, переименование)
  requirements.txt     — зависимости
  templates/
    index.html         — интерфейс
