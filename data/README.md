# Данные SK-Reporter

| Папка | Раздел UI | Содержимое |
|-------|-----------|------------|
| `templates/` | Отчётность → `/daily` | Болванки подрядчиков |
| **`luvr/`** | **Планирование → ЛУВР** (первая карточка) | ЛУВР.xlsx, luvr.yaml, шаблоны Прил.7 и расстановки (xlsm) |
| `projects/{id}/` | Планирование → Проекты | ВОР, project.yaml, vor.json |
| `personnel/` | Планирование → Сотрудники | Исходный Excel для импорта в PostgreSQL |
| `tk/` | Планирование → ОТКК | Файлы .doc на сервере; каталог (id ↔ файл) — PostgreSQL |
| `engineer/` | — | docx-шаблон отчёта инженера |

Подробно про ЛУВР: [`luvr/README.md`](luvr/README.md).

Тяжёлые бинарники (pdf, doc, xlsx, xlsm) по умолчанию **не в git** — см. `.gitignore`.  
На офисном ПК после `git pull` кладите файлы в те же пути.
