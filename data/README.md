# Данные SK-Reporter

| Папка | Раздел UI | Содержимое |
|-------|-----------|------------|
| `templates/` | Отчётность → `/daily` | Болванки подрядчиков |
| `projects/{id}/` | *(в разработке)* | Архив ВОР (docx, vor.json) — без yaml-метаданных |
| `personnel/` | Планирование → Сотрудники | Исходный Excel для импорта в PostgreSQL |
| `tk/` | — | Каталог технологических карт (файлы) |
| `engineer/` | Инженер ФИО | docx-шаблон отчёта инженера |

Справочники **Сотрудники** и **ОТКК** — в PostgreSQL (RelaxDev).

Тяжёлые бинарники (pdf, doc, xlsx, xlsm) по умолчанию **не в git** — см. `.gitignore`.
