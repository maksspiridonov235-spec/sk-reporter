# Данные SK-Reporter

| Папка | Раздел UI | Содержимое |
|-------|-----------|------------|
| `templates/` | Отчётность → `/daily` | Болванки подрядчиков |
| `normative/` | Предписания | Локальная база НД (`manifest.yaml` + `texts/`) |
| `engineer/` | Инженер ФИО | docx-шаблон отчёта инженера |
| `projects/{id}/` | *(опц.)* | `work_tk_map.yaml` — сопоставление работ и ОТКК |

Справочники **Сотрудники**, **Подрядчики**, **Проекты**, **ОТКК** — в PostgreSQL (RelaxDev).

Тяжёлые бинарники (pdf, doc, xlsx, xlsm) по умолчанию **не в git** — см. `.gitignore`.
