# Данные SK-Reporter

| Папка | Раздел UI | Содержимое |
|-------|-----------|------------|
| `templates/` | — | Болванки подрядчиков (`/daily`) |
| `luvr/` | Главная → **ЛУВР** (`/luvr`) | Лист учёта времени |
| `projects/{id}/` | Планирование → Проекты | ВОР, project.yaml, vor.json |
| `personnel/` | Планирование → Сотрудники | Справочник, personnel.yaml |
| `tk/` | Планирование → ОТКК | Технологические карты, manifest.yaml |
| `engineer/` | — (шаблон отчёта) | docx для bat инженера |

Тяжёлые бинарники (pdf, doc, xlsx) по умолчанию **не в git** — см. `.gitignore`.
На рабочей машине кладите файлы в эти же пути после `git pull`.
