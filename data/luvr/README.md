# data/luvr — планирование по времени

Центральный контур планирования SK-Reporter. Подробный план — `docs/memory.md` → «СЕЙЧАС В РАБОТЕ».

## Файлы на диске (офисный ПК)

| Файл | Назначение |
|------|------------|
| `ЛУВР.xlsx` | Лист учёта времени — **источник правды** для посещений по дням |
| `luvr.yaml` | Кэш для сайта (в git); собирается из xlsx |
| `Прил 7 … (макр).xlsm` | Шаблон **Приложения 7** (время оказания услуг), с макросами |
| `Отчёт-расстановка … (макр).xlsm` | Шаблон **расстановки**, с макросами |

Имена шаблонов — в `manifest.yaml`.

## Сборка кэша

```bash
python scripts/build_engineer_data.py --luvr
```

После правок в Excel или на сайте — пересборка и/или экспорт в xlsx (когда будет в UI).

## Git

- **В репо:** `luvr.yaml`, `manifest.yaml`, этот README  
- **Не в репо:** `*.xlsx`, `*.xlsm` (локально на офисном ПК)
