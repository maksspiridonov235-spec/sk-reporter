# Болванки шаблонов (.docx)

Папка с Word-шаблонами для сборки сводных отчётов по компаниям.  
**В веб-интерфейс не загружаются** — сервер читает файлы отсюда при «Подготовить» и «Сформировать».

## Обязательные файлы

- **`Ежедневный отчет Шаблон.docx`** — эталон сетки таблицы для макроса подготовки
- По одному файлу на компанию — имя должно содержать ключевое слово из `companies.py` (например `ЮНС_`, `Евракор_`, `НГСК_`)

## Git

Шаблоны **хранятся в репозитории** (`data/templates/*.docx`). Остальные `.docx` (отчёты, output) в git не попадают.

## Первичная установка на офисном ПC

Если после `git pull` папка пуста или сервер пишет «Папка с болванками не найдена»:

1. Скопируйте `.docx` из резервной копии или с другого ПC в `data/templates/`
2. Либо выполните полный клон репозитория с LFS/файлами (если болванки были в коммите)

Подробнее: [docs/DATA_TEMPLATES.md](../docs/DATA_TEMPLATES.md)

## Правило

**Не вырезать шаблоны из этой папки** — только копировать при обновлении. Программа меняет **содержимое** файлов (дату в тексте), имена на диске при «Подготовить» не меняются.
