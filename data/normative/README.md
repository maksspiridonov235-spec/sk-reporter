# Локальная база нормативных документов

Используется модулем **Предписания**: перед Ollama система ищет текст НД здесь (без Техэксперт и интернета).

## Как добавить документ

1. Положите текст документа (или нужные пункты) в `texts/`, например `texts/519.txt` — кодировка UTF-8.
2. Добавьте запись в `manifest.yaml`:

```yaml
documents:
  - id: rostekhnadzor-519
    title: "Приказ Ростехнадзора от 11.12.2020 N 519"
    kind: Приказ
    number: "519"
    date: "11.12.2020"
    issuer: Ростехнадзор
    file: texts/519.txt
```

3. Перезапуск сервера не обязателен — manifest читается при каждой проверке.

## Сопоставление с B19

По полям `number`, `kind`, `issuer`, `date` из ячейки B19. Чем точнее запись в manifest — тем надёжнее поиск.
