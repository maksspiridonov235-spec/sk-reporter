# Шаблон ежедневного отчёта инженера

Сюда кладётся **пустой или образцовый** docx в том же формате, что инженеры загружают в «Ежедневные отчёты» (`/daily`).

Путь по умолчанию: `data/engineer/report_template.docx` (копия «Ежедневный отчет (шаблон).docx»).

Форма в UI: `/engineer-hub/daily-report` — поля как в шаблоне.
