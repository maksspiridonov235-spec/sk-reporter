# Запуск сервера

## Шаг 1: Откройте PowerShell

## Шаг 2: Перейдите в папку веб-приложения

```powershell
cd C:\Users\Anton\Desktop\sk-reporter\webapp
```

## Шаг 3: Активируйте виртуальное окружение

```powershell
.\venv\Scripts\Activate.ps1
```

Вы должны увидеть `(venv)` в начале строки в PowerShell.

## Шаг 4: Запустите сервер

```powershell
python -m uvicorn main:app --reload --port 8000
```

Вы должны увидеть:
```
INFO:     Uvicorn running on http://127.0.0.1:8000
INFO:     Application startup complete
```

## Шаг 5: Откройте браузер

Перейдите по адресу: **http://localhost:8000**

## Использование

1. **Загрузите отчёты** (левая панель):
   - Нажмите на область загрузки "Отчёты от инженеров"
   - Выберите .docx файлы с ежедневными отчётами

2. **Загрузите шаблоны** (левая панель):
   - Нажмите на область загрузки "Шаблоны подрядчиков"
   - Выберите .docx файлы шаблонов (по одному на компанию)
   - Имена файлов должны содержать ключевые слова компаний

3. **Сформируйте отчёты**:
   - Кликните на название компании (кнопки в центре)
   - ИЛИ нажмите "СФОРМИРОВАТЬ ВСЕ ОТЧЁТЫ"

4. **Скачайте результаты**:
   - Результаты появятся в разделе "Готовые файлы"
   - Нажмите "Скачать" рядом с файлом

## Остановка сервера

В PowerShell нажмите: **Ctrl + C**

## Если что-то не работает

- Проверьте что сервер запущен (строка "Uvicorn running...")
- Перезагрузите страницу браузера (F5)
- Посмотрите логи в PowerShell - там будут ошибки
